# Line-Coding
Visualize NRZ, NRZI, RZ, Manchester, differential base, 4B-5B, AMI 50% and 100%, HDB-3, MLT-3

## ⚙️ Setup 
* Be sure to have python 3+ installed
### Libraries
Install the following libraries: `Pillow`, `Matplotlib`.
```
pip install pillow matplotlib
```

<br>

## 💻 Use
* Clone the repo, than run `main.py`:

```
python3 main.py
```
